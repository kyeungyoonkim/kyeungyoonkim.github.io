Accessibility Web Summarizer

Dec 2024 · A web summarization tool designed for clarity, structure, and accessibility.

What it does

This project summarizes web pages into a clean, scannable brief that prioritizes readability and accessible structure (clear headings, concise paragraphs, and reduced cognitive load).

Accessibility-first goals

  • Structured output that is easy to navigate with assistive technologies
  • Short, informative paragraphs with minimal fluff
  • Focus on key takeaways while preserving context
