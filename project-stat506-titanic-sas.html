STAT 506 Final: Titanic Analysis (SAS)

Fall 2024 · Purdue STAT 506 — Base SAS programming applied to the classic Titanic passenger dataset.

Summary

Course final project that walks through a full SAS workflow on the Titanic dataset (CSV): import and metadata review, controlled listing and subsetting, DATA-step derivations, sorting with group logic (RETAIN, sum statement, FIRST./LAST.), character and concatenation functions, SQL extracts and sorts, user-defined permanent formats with PROC FREQ, transposition for reshaped reporting, documented titles and footnotes with macro variables, and export to CSV, another format, and Excel via LIBNAME.

Data

Public Titanic passenger data (survival, class, demographics, fare, cabin, embarkation port, etc.), commonly used for teaching and on platforms such as Kaggle. Variables include PassengerID, Survived, Pclass, Name, Sex, Age, SibSp, Parch, Ticket, Fare, Cabin, and Embarked.

What the write-up demonstrates

  • Ingestion & documentation: PROC IMPORT, PROC CONTENTS (VARNUM)
  • Reporting basics: PROC PRINT with OBS=, VAR, WHERE (IN/LIKE), multiple formats
  • DATA step: SET, WHERE, DROP/KEEP, FORMAT, LENGTH, conditional assignments for new variables
  • By-group processing: PROC SORT, RETAIN, sum statement, FIRST./LAST. in conditionals
  • Character tools: SUBSTR, LENGTH, SCAN, PROPCASE, CATX (and related)
  • PROC SQL: create reduced tables, WHERE subsets, ORDER BY two columns
  • Formats & frequency: VALUE formats (character and numeric), permanent catalog, PROC FREQ
  • Reshape: PROC TRANSPOSE (BY, OUT=, NAME=, PREFIX=) with PROC PRINT of result
  • Documentation & delivery: TITLE/TITLE2/FOOTNOTE with macro variables, LABEL, clear titles; PROC EXPORT (CSV + other); Excel via LIBNAME
