STAT 506 Final: Titanic Analysis (SAS)

October 2024 · Purdue STAT 506 — Hands-on Base SAS through a standard teaching dataset (CSV).

📘 SAS programming skills

Titanic dataset: implementing core SAS data-handling features end to end.

Techniques covered

  • ✓ Data I/O — PROC IMPORT / PROC EXPORT, LIBNAME (including Excel)
  • ✓ Data manipulation — DATA step, SET, WHERE, KEEP / DROP
  • ✓ Character functions — SUBSTR, SCAN, CATX, PROPCASE (and related)
  • ✓ SQL — PROC SQL: SELECT, WHERE, ORDER BY, CREATE TABLE
  • ✓ Sorting & grouped logic — PROC SORT, RETAIN, FIRST. / LAST., sum statement
  • ✓ User-defined formats — PROC FORMAT, VALUE (with PROC FREQ)
  • ✓ Restructuring — PROC TRANSPOSE (BY, OUT=, NAME=, PREFIX=)
  • ✓ Reporting — TITLE, FOOTNOTE, LABEL, PROC PRINT (including macro variables in titles)

Project overview

The Titanic passenger dataset (891 observations, 12 variables) is a common benchmark for introductory analytics. This final project uses it to practice the main SAS programming patterns required in the course: moving data from flat files into SAS, shaping and enriching tables, summarizing and printing results, and writing outputs back out in multiple formats.

What I implemented

  • An end-to-end pipeline: import CSV → clean and subset → create new variables → export results.
  • Parallel use of DATA step and PROC SQL where each approach fits best.
  • Formal coverage of ten syllabus topics (topics 1, 3, 5, 7, 10, 12, 15, 18, 22, and 28), including PROC CONTENTS, controlled PROC PRINT, formats, PROC SQL queries, BY-group processing, character functions, permanent formats, PROC TRANSPOSE, and documented listings.

Deliverables

  • titanic.sas — single program containing the full SAS code for the assignment.
  • STAT506_final_YoonKim.pdf — write-up with log snippets, procedure output, and narrative (course submission).
